# BrawlCratePlugins
Plug-ins for BrawlCrate using BrawlAPI. Includes scripts for exporting mass info to text files, and some other helpful automated bits.

Feedback always welcome @ mawwwk#1068

Plug-ins:
- Export ASL Info (P+)
- Export Param Info (P+)
- Export Tracklist Info (P+)

Loaders:
- Stock Icon Exporter
